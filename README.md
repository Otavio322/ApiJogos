--> 🃏 Cartas de Magic

Projeto web simples que permite buscar cartas de Magic: The Gathering usando a API pública do Scryfall.

---

--> Funcionalidades

* 🔍 Buscar cartas pelo nome
* 🎲 Gerar carta aleatória
* 🖼️ Exibir imagem da carta
* 📱 Layout responsivo (mobile e desktop)

---

--> Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* API Scryfall
